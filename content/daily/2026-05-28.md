# 日报 2026-05-28

1. 智谱创始人唐杰：2026年AI突破焦点是长时程Agent，自改进模型最有前景
   来源: 智谱AI/机器之心
   智谱AI创始人唐杰教授在X上发文预判五大AI趋势：2026年LLM主要突破将从"推理能力"转向"长时程任务（Long-Horizon Tasks）"；一人公司将向无人公司演进；自主Agent系统（AAS）将成为下一个前沿方向；最困难但最有前景的路径是模型"自我进化"。此观点与红杉资本、LangChain CEO的预测一致，OpenAI Codex /Goal 和 Claude Code Computer Use 等功能已在印证这一方向。

2. 国产Agent集体突破：Qwen3-Max-Thinking、Kimi K2.5 密集发布
   来源: 第一财经/国信证券
   2026年国产大模型竞争全面转向Agent能力。阿里发布千问旗舰推理模型 Qwen3-Max-Thinking，万亿参数规模，36T Tokens 预训练，原生Agent能力可自主调用多种工具；Kimi K2.5 实现"集群式作战"，可调度100个专业分身并行处理1500次工具调用；DeepSeek 开源 DeepSeek-OCR 2 并上线专家模式。数据显示2025年我国企业级AI Agent市场规模已达126亿元，同比增长321.7%。

3. ICML 2026: NSI框架——从交互轨迹中归纳可执行技能程序
   https://arxiv.org/abs/2605.01293
   南京大学等团队提出 Neuro-Symbolic Skill Induction (NSI)，将Agent的交互轨迹提升为模块化、逻辑驱动的程序。传统方法将经验蒸馏为无状态参数化脚本，无法捕捉动态环境中的条件逻辑；NSI 通过合成显式控制流和动态变量绑定，使Agent能够从少量示例中归纳技能并泛化到未见过的目标。实验在多项Agent任务上超越现有基线，被 ICML 2026 接收。

4. 安全研究：被越狱的前沿模型仍保留完整能力
   https://arxiv.org/abs/2605.00267
   Daniel Zhu 等人的研究发现，前沿LLM在被越狱后不会显著丧失原有能力。此前普遍认为安全对齐与模型能力存在权衡——越狱会损害性能；该研究通过系统性实验表明，被成功越狱的模型在标准基准上表现与原始模型几乎一致，提示当前安全机制存在根本性脆弱性，仅靠对齐训练不足以防止恶意滥用。
